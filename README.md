# LiDAR Simulator

A personal project to simulate airborne LiDAR data collection and visualize it in real-time using OpenGL.

## Structure
- **Simulator** – Generates synthetic LiDAR data using DEM terrain.
- **Visualizer** – Renders point clouds in real-time using OpenGL.
- **PythonTools** – Optional data analysis and visualization utilities.

## Tech Stack
C++17, OpenGL, GDAL, Eigen/GLM, Visual Studio
